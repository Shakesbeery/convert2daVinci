Files best used with:

https://www.youtube.com/watch?v=UiF1Jt65RYk

Usage:

-f specifies the gcode file you want to convert
-c indicates using cura header
-a indicates if you want to autocalculate the # of layers and filament length used (in the video they are manually calculated and changed)

example: $python gcode_to_davinci.py -f ./my_file.gcode -c True -a True
